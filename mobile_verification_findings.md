# Mobile verification findings

Date: 2026-06-13

## Summary

The mobile optimisation pass was implemented and verified on the following pages using fresh iPhone-sized full-page screenshots:

- Home
- Guides index
- Schools
- Jobs
- Stories
- About
- BA Speedbird Academy guide

## Verified improvements

### Global/mobile layout

- Section spacing is tighter and more consistent on mobile.
- Container gutters feel balanced and no longer overly compressed.
- Hero sections now use smaller headings and reduced vertical padding.

### Home page

- Hero is shorter and better proportioned on mobile.
- Cards and section blocks have tighter padding.
- The page feels more scannable from top to bottom.

### Schools page

- Hero spacing and heading scale are improved.
- Filter controls stack more naturally on small screens.
- School cards have reduced padding and read more cleanly.

### Guide pages

- Guide hero sections are tighter.
- Metadata wraps more cleanly.
- Article and CTA blocks have improved mobile padding.

### Jobs page

- Hero and intro content fit mobile more cleanly.
- Job cards are easier to scan.
- Overall density is improved without looking cramped.

### Stories page

- Hero and body spacing improved.
- Story header cards and Q&A blocks are more comfortable on mobile.

### About page

- Hero typography is more balanced.
- Founder panel stacks better on mobile.
- Founder card uses bottom border separation instead of desktop-style right border.

### Guides index

- Hero is tighter.
- Category spacing is improved.
- CTA block has better mobile padding.

### Footer

- Footer now uses a more mobile-friendly 2-column layout.
- Vertical rhythm and padding are improved.

## Outcome

The mobile experience is now materially closer to the desktop quality level. The site remains the same site and content, but the responsive presentation is more polished, readable, and conversion-friendly on phones.

## Remaining note

A real founder photo would still improve the About page and author trust signals further, but this is not a mobile blocker.
